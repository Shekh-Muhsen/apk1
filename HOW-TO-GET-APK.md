# How to get your APK — 3 steps, no software needed

## Step 1 — Put your files on GitHub (free)

1. Go to github.com → Sign up (free)
2. Click the + button → "New repository"
3. Name it: ocean-adventure
4. Click "Create repository"
5. Click "uploading an existing file"
6. Drag ALL files from C:\Users\USER\Documents\DDA\ into the upload box:
   - ocean-adventure-yacht.html
   - android-build/ folder
   - .github/ folder
7. Click "Commit changes"

## Step 2 — GitHub builds your APK automatically

- Go to your repo page
- Click the "Actions" tab
- You will see "Build Ocean Adventure APK" running (takes 3-5 minutes)
- Wait for the green checkmark ✅

## Step 3 — Download your APK

1. Click on the completed workflow run
2. Scroll down to "Artifacts"
3. Download:
   - OceanAdventure-release → APK file (install directly on phone)
   - OceanAdventure-playstore-aab → AAB file (upload to Play Store)

## Upload AAB to Play Store

1. play.google.com/console → pay $25 one-time
2. Create app → "Ocean Adventure"
3. Production → Create new release → Upload the .aab file
4. Add screenshots (take them on your phone)
5. Fill privacy policy: use privacypolicygenerator.info (free)
6. Submit → approved in 1-3 days
